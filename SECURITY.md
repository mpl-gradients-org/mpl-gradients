# Security Policy

## Supported Versions

Since this project is in active development, we currently only support the latest available version.

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |
| Older   | :x:                |

## Reporting a Vulnerability

We take the security of `mpl-gradients` seriously. If you discover a security vulnerability, please do **not** report it as a public issue.

Instead, please send an email to **[sharmasanchitrishi@gmail.com]**.

### What to include in your report:
* A description of the vulnerability.
* Steps to reproduce the issue.
* Any relevant code snippets or screenshots.

We will try to acknowledge your report within 48 hours and will keep you updated on the progress of a fix.
